# THM-Digital-Forensics-Fundamentals
עולם המרתק של ה-Digital Forensics (פורנזיקה דיגיטלית). זהו תחום שמשלב עבודת בילוש קלאסית עם טכנולוגיה מתקדמת, שבו המטרה היא "להנשים" נתונים ולגרום להם לספר מה באמת קרה.  להלן ריכוז היסודות (Fundamentals) שכל חוקר מתחיל חייב להכיר:

תהליך הפורנזיקה הדיגיטלית (The Process)
כדי שראיה תהיה קבילה בבית משפט, חוקרים עוקבים אחר מודל סטנדרטי (לעיתים נקרא NIJ Guide או ISO/IEC 27037). השלבים המרכזיים הם:

1. איסוף ורכישה (Acquisition)
זהו השלב הקריטי ביותר. המטרה היא להוציא את הנתונים מהמכשיר מבלי לשנות אותם אפילו בביט אחד.

Write Blocker: כלי חומרה או תוכנה שמונע מהמחשב "לכתוב" נתונים לכונן הנבדק (למשל, עדכון זמן גישה אחרון לקובץ).

Imaging: יצירת עותק זהה (Bit-stream Image). לא מדובר ב-Copy-Paste רגיל, אלא בשכפול של כל המגזרים (Sectors) בכונן, כולל שטח ריק שבו עשויים להימצא קבצים שנמחקו.

2. אימות (Authentication)
איך מוכיחים שהעותק שיצרנו זהה למקור? באמצעות פונקציות Hashing.

מריצים אלגוריתם (כמו SHA-256) על הכונן המקורי ועל העותק.

אם התוצאה (ה-Hash) זהה ב-100%, סימן שהראיה "טהורה" ולא זוהמה בתהליך.

3. ניתוח (Analysis)
כאן מתחילה עבודת הבלש. החוקר מחפש עקבות (Artifacts) במקומות סמויים:

File Carving: טכניקה לשחזור קבצים שאין להם רישום במערכת הקבצים (קבצים שנמחקו או שחלק מהם נדרס).

Metadata: מידע על המידע (למשל: מי יצר את המסמך, באיזו מצלמה צולמה התמונה והאם היו בה נתוני GPS).

Timeline Analysis: בניית ציר זמן של אירועים כדי להבין מה קדם למה.

4. דיווח (Reporting)
הדו"ח הסופי חייב להיות ברור גם לאנשים שאינם טכנולוגיים (שופטים, עורכי דין). הוא צריך לפרט את הכלים שבהם נעשה שימוש, את הצעדים שננקטו ואת הממצאים שנמצאו.

מושג מפתח: סוגי הנתונים
במהלך המשימה הזו, תלמד להבדיל בין שני סוגי נתונים:

Volatile Data (נתונים נדיפים): מידע שנמחק ברגע שמכבים את המחשב (כמו ה-RAM). אלו נתונים שחייבים לאסוף בזמן שהמחשב דלוק.

Non-Volatile Data (נתונים לא נדיפים): מידע שנשמר על הכונן הקשיח גם ללא חשמל.

ארבעת שלבי הפורנזיקה הדיגיטלית (לפי NIST)
תהליך עבודה מובנה הוא קריטי כדי שהראיות יהיו קבילות בבית משפט.

איסוף (Collection):

זיהוי כל המכשירים בזירה (מחשבים, USB, מצלמות).

הבטחה שהנתונים המקוריים לא ישתנו (שימוש בחוסמי כתיבה).

תיעוד קפדני של כל פריט שנאסף.

בדיקה (Examination):

סינון (Filtering) הררי המידע שנאספו.

חילוץ נתונים רלוונטיים בלבד (למשל: רק קבצים מתאריך מסוים או משתמש ספציפי) כדי לייעל את המשך החקירה.

ניתוח (Analysis):

הצלבת ראיות (Correlation) ממקורות שונים.

בניית ציר זמן כרונולוגי של האירועים והסקת מסקנות מהממצאים.

דיווח (Reporting):

הכנת דו"ח מפורט הכולל את המתודולוגיה והממצאים.

הכללת תקציר מנהלים המותאם לאנשים שאינם טכנולוגיים (שופטים, הנהלה).

סוגי התמחויות בפורנזיקה דיגיטלית
מכיוון שכל סביבה טכנולוגית דורשת כלים אחרים, התחום מתחלק לתתי-התמחויות:

מחשבים (Computer): התחום הנפוץ ביותר, חקירת תחנות עבודה ולפטופים.

מכשירים ניידים (Mobile): חילוץ הודעות, שיחות ומיקומי GPS מסמארטפונים.

רשתות (Network): ניתוח תעבורת רשת ויומני שרתים (Logs) כדי להבין תנועה בין מכשירים.

מסדי נתונים (Database): חקירת פריצות למאגרי מידע ושינוי נתונים רגישים.

ענן (Cloud): חקירת תשתיות ענן (מורכב יותר בשל הגישה המוגבלת לתשתית הפיזית).

דוא"ל (Email): חקירת תכתובות לאיתור הונאות, פישינג או דליפת מידע.
